# 💰 Sistema Bancário em Java

## 📌 Descrição
Projeto desenvolvido em Java simulando um sistema bancário com operações básicas.

## 🚀 Funcionalidades
- Criação de contas
- Depósito
- Saque
- Consulta de saldo

## 🛠 Tecnologias utilizadas
- Java
- Programação Orientada a Objetos (POO)

## 📂 Estrutura do projeto
- Banco.java
- Conta.java
- ContaCorrente.java
- ContaPoupanca.java
- CaixaEletronico.java
